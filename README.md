# RailsMVPStarter

The goal of this project is to have a simple Rails app that developers can start with to create products faster.  Building tooling like authentication and payments is so last decade (century??).

This project aims to provide a base line of system level services so you start with some plumbing you'd rather not waste time writing over and over again.

## Features

1.  Authentication
2.  Signup with Google
3.  Friendly ID for nice URLs
4.  Simple Stripe Implementation
5.  Bootstrap  4 (via CDN)
